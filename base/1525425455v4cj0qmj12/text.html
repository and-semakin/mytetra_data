bash:

1. Для того, чтобы добавить переменную окружения во все сессии оболочки bash, нужно дописать в файл ~/.bashrc:

export DISPLAY=0


2. Для того, чтобы переменная добавлялась в момент логина в систему, нужно дописать в файл ~/.profile:

export DISPLAY=0


3. Для того, чтобы переменная была доступна оконным приложениям, нужно дописать в файл /etc/environment:

DISPLAY=0 # без export


Чтобы обновить список переменных окружения:

$ source ~/.bashrc
$ source ~/.profile
